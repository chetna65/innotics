HACK PACK
PROJECTS
Automatic Plant Watering System
The Automatic Plant Watering System is a smart solution that ensures plants receive the right amount of water without human intervention. It uses an Arduino UNO, a soil moisture sensor, and a water pump to monitor soil dryness and automatically water the plant when needed.
The Smart Blind Stick
The Smart Blind Stick is an assistive device designed to help visually impaired individuals navigate safely. It uses an Arduino UNO, an ultrasonic sensor to detect obstacles, and a buzzer or vibration motor to alert the user when an object is nearby.
LPG Gas Leakage Detection System
The LPG Gas Leakage Detection System is a safety device that detects harmful gas leaks in homes, industries, and workplaces. It uses an MQ-6 gas sensor to monitor the air for LPG leaks and triggers an alarm (buzzer) and LED warning if a leak is detected.
Arduino RFID Attendance System
The Arduino RFID Attendance System is a smart way to record attendance using RFID cards or tags. Each user gets a unique RFID card, which is scanned by an RFID reader connected to an Arduino UNO. When a card is scanned, the system logs the attendance and can display the details on an LCD screen or store data in a database.
Automatic Railway Gate
The Automatic Railway Gate Control System is a smart solution that helps improve railway safety by automating the opening and closing of gates at level crossings. Using ultrasonic sensors, the system detects approaching trains and automatically lowers the gate.
Car Parking System
The Car Parking System using Arduino is a smart automation project that helps manage parking spaces efficiently. It uses an ultrasonic sensor to detect the presence of a car and an LCD display to show available parking slots.
Automatic Exhaust Fan
The Automatic Exhaust Fan system is designed to improve air quality by automatically turning on the fan when it detects high temperature or poor air quality. It uses a DHT11 sensor to measure temperature and humidity and an MQ gas sensor to detect smoke or harmful gases.
Automatic Smart Street Light
The Automatic Smart Street Light system is designed to save electricity by automatically turning street lights ON in darkness and OFF in daylight. It uses an LDR (Light Dependent Resistor) sensor to detect ambient light levels.
Clap Light
The Clap Light system allows you to turn a light ON or OFF simply by clapping. It uses a sound sensor to detect clap sounds and an Arduino to process the signal. When the system detects a clap, it toggles a relay module, which controls a light bulb or LED.
